# Netflix Data Analysis

## Project Overview
This project performs an exploratory data analysis (EDA) on the Netflix
Movies and TV Shows dataset to identify content trends, genre popularity,
and country-wise distribution using Python.

The analysis focuses on understanding Netflix’s content strategy through
data cleaning, aggregation, and visualization techniques.


## Dataset
- **Source:** Kaggle – Netflix Movies and TV Shows Dataset  
- **Total Records:** ~8,800  
- **Total Features:** 12  


## Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  


## Key Analysis Performed
- Distribution of Movies vs TV Shows on Netflix  
- Country-wise content production analysis  
- Year-wise growth of Netflix content  
- Identification of most common genres  
- Analysis of movie duration distribution  


## Visualizations
- Bar charts for categorical comparisons  
- Line plots for content growth trends  
- Histograms for duration analysis  
- Count plots for rating and type distributions  


## Key Insights
- Netflix hosts a higher number of Movies compared to TV Shows  
- The United States and India are the top content-producing countries  
- A significant increase in content production is observed after 2015  
- Drama is the most frequently listed genre on Netflix  


## Conclusion
This project demonstrates the use of exploratory data analysis techniques
to extract meaningful insights from real-world data. It highlights how
data visualization and statistical analysis can be used to understand
business content strategies effectively.
